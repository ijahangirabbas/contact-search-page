# Contact Search App

A lightweight single-page web application for searching contacts and recording interest/comments.

## Features

- Live search by first or last name
- Partial string matching
- Mobile-friendly interface
- Interested button
- Comment submission
- Timestamped records
- JSON-based storage

## Tech Stack

- HTML
- CSS
- JavaScript
- Node.js
- Express

## Installation

```bash
npm install
npm start
